REST
- 소프트웨어의 구조를 형성하는 방법에 대한 가이드라인
- 6개 가이드라인을 모두 만족한다면 'RESTful API'
- 메소드
    - GET : 조회
    - POST : 생성
    - PATCH : 업데이트(일부)
    - PUT : 업데이트(전체)
    - DELETE : 삭제
